# Recipe Catalog CMS (Wagtail + Django)

A simple recipe management platform built with **Django** and **Wagtail CMS**, fully containerized using **Docker**.  
The project allows creating and managing recipe pages with structured content such as ingredients, cooking steps, nutrition information, and images.

---

##  Features

- Create and manage recipe pages using Wagtail CMS
- Structured recipe fields:
  - Description
  - Ingredients
  - Cooking method
  - Origin
  - Calories
  - Prep time / Cook time
  - Servings
- Featured image support
- Rich text editing for ingredients and steps
- Admin-friendly CMS interface (Wagtail)
- Dockerized development environment

---

##  Tech Stack

- Python 3.12
- Django
- Wagtail CMS
- SQLite (development)
- Docker & Docker Compose
